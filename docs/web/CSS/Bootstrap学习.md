# Bootstrap 学习

> Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。

## 1. 响应式开发

为了便于理解 Bootstrap 的响应式布局原理，我们先简要了解响应式开发，就是使用媒体查询针对不同宽度的设备进行布局和样式的设置，从而适配不同的设备

|         设备划分         |     尺寸区间      |
| :----------------------: | :---------------: |
|     超小屏幕（手机）     |      <768px       |
|     小屏设备（平板）     | >=768px ~ <992px  |
|  中等屏幕（桌面显示器）  | >=992px ~ <1200px |
| 宽屏设备（大桌面显示器） |      >1200px      |
