# React 基础

React 是一个用于构建用户界面的 JavaScript 库

React 的三个特点：

  • 声明式
  • 组件化
  • 一次学习，随处编写

# 在浏览器环境中引入 react

我们需要引入 react 核心库，react-dom 扩展库以及 babel。 可以下载到本地引入，以下为了方便使用了 cdn 引入。

<script src="https://cdn.bootcdn.net/ajax/libs/react/18.2.0/umd/react.development.js"></script>
<script src="https://cdn.bootcdn.net/ajax/libs/react-dom/18.2.0/umd/react-dom.development.js"></script>
<script src="https://cdn.bootcdn.net/ajax/libs/babel-standalone/7.19.3/babel.min.js"></script>

值得注意的是，当我们在 bootcdn 等 cdn 网站上搜索 react,react-dom,babel 这三个包时，会有很多的版本，选择错误的版本会导致 react 无法在浏览器中运行，挫败我们学习 react 的信息。

比如我们在 cdn 官网上也会找到这个包 https://cdn.bootcdn.net/ajax/libs/react/18.2.0/cjs/react.development.js，

对比一下前面引入的 react 核心库链接，我们会发现其实就是 cjs 和 umd 的区别。cjs 也就是 commonjs，umd 则是 universal module definition，前者适用于 nodejs 环境，后者则前后端环境都适用。

再比如，搜索 babel 时会出现 babel-core，bable-standalone 等库，为了适用于浏览器环境，我们要适用 babel-standalone。

# 虚拟 DOM

React 使用了虚拟 DOM 和 Diff 算法来帮助我们快速修改 DOM。

虚拟 DOM：

  • 本质是 Object 类型的对象（一般对象）。
  • 比真实 DOM“轻”很多，因为虚拟 DOM 是在 React 内部使用，无需真实 DOM 的那么多属性。
  • 虚拟 DOM 会被 React 转换为真实 DOM 并呈现到页面上。

# JSX

JSX 是 Javascript XML 的缩写，是 react 定义的一种类似于 XML 的 JS 扩展语法，上文中使用 babel 的目的就是解析这种语法。

JSX 本质是 React.createElement(component, props, ...children)方法的语法糖，可以简化我们创建虚拟 DOM。

JSX 语法不是一个字符串，也不是一个标签，表面上看起来像标签，实际上会得到一个虚拟 DOM 的对象，当然，前面谈过，虚拟 DOM 对象并没有那么高大上，也就是一个普通的对象。

这是一个 JSX 语法的例子：

const name = "刻晴";
const ele = <h1 id="keqing">Hello, {name}</h1>;

2

JSX 语法的一些注意点：

  • 只能有一个根标签
  • 标签必须闭合
  • 在{}中可以使用任何 js 语法，这有助于我们通过 js 传递数据。
  • 原 html 标签的 class 属性名现在要写成 className，原 label 标签的 for 属性名现在要写成 htmlFor。
  • 对于多单词的属性名，要使用小驼峰命名法，如原 html 标签中的 onclick 要写成 onClick。
  • 行内样式 style 的值不能是字符串，而要是对象。即要写成双大括号的形式。
  • 小写字母开头的标签 react 会解释为 html 中的标签，大写字母开头的标签 react 会解释为组件。

# PropTypes

PropTypes 可以对 props 中的属性进行类型和必要性的限制，同时还可以通过 defaultProps 来设置默认值。

需要导入 prop-types 库，如浏览器中可以使用如下 cdn 导入： <script src="https://cdn.bootcdn.net/ajax/libs/prop-types/15.8.1/prop-types.min.js"></script>

PropTypes 使用示例：

class Person extends React.Component {
  render() {
    return (
      <h2>
        {this.props.name}：{this.props.age}
      </h2>
    );
  }

  static propTypes = {
    name: PropTypes.string.isRequired,
    age: PropTypes.number,
  };

  static defaultProps = {
    name: "张三",
    age: 18,
  };
}

const root = ReactDOM.createRoot(document.getElementById("root"));
root.render(<Person name="李四" age={12} />);
