# Git 基本使用

Git 下载不必多说，在https://git-scm.com/ (opens new window)下载安装即可。

# 一.配置

  1. 全局配置用户名和邮箱
git config --global user.name "zhangchuangxin"
git config --global user.email "2911331070@qq.com"
# 查看配置
git config --list
# 可以直接查看某个配置项
git config user.name
git config user.email

  1. 本地配置用户名和邮箱

执行 git init 命令后会初始化 git 仓库，这表示 git 开始管理你的项目。 此时当前目录下会出现一个隐藏文件夹.git。.git 文件夹下的 config 文件则是项目的本地配置。

# 类似全局配置，不要--global即可
git config user.name "zhangchuangxin"
git config user.email "2911331070@qq.com"
# 查看本地配置
vim .git/config

vim 的知识在次不多说，简要介绍一下怎么退出（初次使用 vim 时可能会遇到不会退出的尴尬） 输入:q 或者 ZZ 即可退出

如果你电脑中配置了 vscode 的环境变量，使用 code .git/config 同样可以打开项目的本地配置文件

  1. 更改 Git 默认使用的文本编辑器

Git 默认使用的文本编辑器一般可能会是 Vi 或者 Vim，你也可以更改为自己喜欢的编辑器

git config --global core.editor code
1
  1. 差异分析工具

还有一个比较常用的是，在解决合并冲突时使用哪种差异分析工具。比如要改用 vimdiff 的话

git config --global merge.tool vimdiff
1

TIP

git 命令行窗口中内容太多想清理怎么办？输入 clear 命令或者使用快捷键 ctrl + l
